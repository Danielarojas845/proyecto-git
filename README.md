# Proyecto Git - Ejercicio

Este proyecto consiste en practicar el flujo básico de Git y GitHub mediante la creación de ramas, modificaciones y fusiones entre ramas.

## Funcionalidades implementadas
- **Navbar:** Implementado en la rama `feature/navbar`.
- **Footer:** Implementado en la rama `feature/footer`.

## Ramas utilizadas
- `main` — Rama principal.
- `feature/navbar` — Barra de navegación.
- `feature/footer` — Pie de página.

## Descripción
Se trabajó de forma individual siguiendo un flujo Git con múltiples ramas y Pull Requests.
